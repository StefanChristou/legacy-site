New York en Pyjamarama interactive

This prototype web application is a work in progress exploring the use of the high-resolution display on the ipad to show a children's story. The story itself is highly-interactive, with a transparency allowing the book's illustrations to move.

It's the perfect bedtime story. And it features a lovely set of content with which to further explore interactivity on touch devices. The book is by Michaël Leblond and Frédérique Bertrand. It is published by ROUERGUE from Arles, France.

This is a prototype web application made using HTML5 and CSS3 and Javascript (plus Tumult Hype for opening sequence animations and timing). The application is a fun little test project for me as I continue to explore the interactive possibilities of content on touch devices.
